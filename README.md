Welcome to the source code repository for my professional email development portfolio. 

As an Email Developer, I specialize in translating static UI/UX designs into bulletproof, responsive HTML/CSS and MJML templates. I solve complex rendering issues across 50+ major email clients (including legacy Outlook versions) and ensure strict deliverability and accessibility standards.

## 🛠️ Tech Stack & Architecture
This portfolio website is custom-built from scratch to be lightweight, lightning-fast, and highly accessible without relying on heavy frameworks.

* **HTML5:** Semantic, accessible markup.
* **CSS3:** Custom variables, CSS Grid, Flexbox, and mobile-first media queries.
* **Vanilla JavaScript:** DOM manipulation for mobile navigation, light/dark mode toggling, and custom image lightboxes.
* **SEO & Metadata:** Fully optimized with JSON-LD schema, Open Graph (OG) tags, and canonical routing.

## ✨ Key Features
* **Native Light & Dark Mode:** Uses CSS variables and `localStorage` to respect user system preferences while allowing manual toggling.
* **Zero Dependencies:** No heavy libraries (like Bootstrap or jQuery) for maximum performance.
* **Responsive Design:** Fluid scaling typography and layout grids tailored for all devices.
* **Secure Contact Routing:** Form data is securely routed via FormSubmit without exposing backend logic.
