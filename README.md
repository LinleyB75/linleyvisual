# LinleyB | Email Developer Portfolio

Welcome to the source code for my professional portfolio website: [linleyb.com](https://linleyb.com). 

I am an Email Developer based in Mauritius, specializing in translating complex UI/UX designs into bulletproof, responsive HTML, CSS, and MJML emails. I actively share this repository to provide complete transparency into my coding practices, architecture, and attention to detail.

## 🛠️ The Tech Stack

This portfolio—and the email campaigns I develop—are built using:
* **HTML5 / CSS3:** Semantic, accessible, and lightweight code.
* **MJML:** Component-driven framework for rapid, cross-client compatible email production.
* **Handlebars:** Templating logic for dynamic, personalized ESP integrations.
* **VML & Ghost Tables:** Targeted Microsoft Outlook rendering fixes.

## 🌟 Site Features & Architecture

I built this site from scratch to demonstrate the same rigorous standards I apply to email development:

* **Fluid-Hybrid Responsiveness:** The site scales flawlessly on mobile without relying on bloated CSS frameworks.
* **Native Dark Mode:** A custom-built, JavaScript-enabled dark mode toggle that respects system-level `prefers-color-scheme` settings.
* **High Performance:** Minified structures, lazy-loaded assets, and optimized routing.
* **SEO & Accessibility:** Fully implemented ARIA roles, semantic `<main>` and `<header>` tags, structured JSON-LD schema, and a strictly managed `sitemap.xml` and `robots.txt`.
